Buttons and other assets

On this page you find the statement buttons and icons that you can use when applying the rights statements online. Please refer to our guidelines for applying the rights statements for more information on how to correctly apply the rights statements online.

Buttons

Below you find SVG and PNG versions of our buttons that are intended for use as links to the rights statements. For each button we provide a version with blue text and a black and white version. You can also download a ZIP file with all button images which also includes a black version with a fully transparent background and a white version with a fully transparent background of each button.
